# Test Repo Projects

This folder contains multiple standalone projects that can be run individually.

## Projects

### Calendar
A simple web-based calendar application for viewing dates in the past and future.

**How to run:**
1. Navigate to the `calendar/` folder
2. Open `calendar.html` in your web browser (drag and drop the file, or use File → Open)
3. Use the Previous/Next buttons to navigate months
4. Click "Today" to return to the current date

**Files:**
- `calendar.html` - Main HTML structure
- `styles.css` - All styling and layout
- `script.js` - Calendar functionality and interactivity

**Note:** All three files must stay in the same folder for the calendar to work properly.

## Adding New Projects

To add a new project:
1. Create a new folder in the `projects/` directory with a descriptive name
2. Place all project files in that folder
3. Add a description to this README
