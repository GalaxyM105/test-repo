# 🎮 Terminal Hangman

A simple command-line Hangman game written in Python.

## How to Play

Run the game with:

```bash
python hangman.py
```

- Guess one letter at a time
- You have **6 attempts** before the hangman is complete
- Try to reveal the hidden word before running out of guesses!

## Features

- 20 built-in tech/programming themed words
- ASCII art hangman that updates with each wrong guess
- Tracks wrong guesses and remaining attempts
- Play again prompt at the end

## Requirements

- Python 3.x (no external libraries needed)
